How this works

The dividers between each heading and description align across all four cards — regardless of title length. No JavaScript, no table layouts, no hacks. Each card uses grid-template-rows: subgrid to participate in the parent grid's row tracks. The tallest title determines the row height for all siblings. Three lines of CSS is all it takes.
